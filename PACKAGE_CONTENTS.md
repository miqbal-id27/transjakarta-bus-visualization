# Package Contents

This package is curated for a GitHub portfolio repository.

Included:

- final D3 web demo under `docs/`, ready for GitHub Pages,
- processed JSON data required by the demo,
- concise executive summary and process documentation,
- R source file for data collection and EDA,
- RMarkdown file for visualization process notes,
- proposal, EDA report, and Five Design Sheets as references.

Excluded:

- raw large CSV files,
- duplicated data files,
- Excel working files,
- assignment brief PDF,
- final implementation report PDF.
